Glosario de Etiquetas HTML

Etiqueta Símbolo Definición
HTML <html> Define la raíz de un documento HTML.
Head <head> Contiene metadatos/información sobre el documento.
Title <title> Especifica un título para el documento.
Body <body> Define el cuerpo del documento.
H1 a H6 <h1> ... <h6> Definen encabezados de diferentes niveles.
Paragraph <p> Define un párrafo.
Anchor <a> Define un hipervínculo.
Image <img> Define una imagen en el documento.
Unordered List <ul> Define una lista desordenada.
Ordered List <ol> Define una lista ordenada.
